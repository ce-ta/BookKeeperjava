# 📚 BookKeeperjava

**BookKeeperjava** は、書籍の登録、編集、削除、検索、並び替えなどの機能を備えた書籍管理アプリケーションです。

## 🛠️ 機能一覧

- ユーザー認証（ログイン／サインアップ）
- 書籍の登録・編集・削除
- ジャンル選択（文学、雑誌、漫画、文庫、文芸書、専門書、児童書・学習参考書）
- 検索・並び替え機能（昇順／降順）
- 多言語対応（日本語／英語）
- Bootstrap によるスタイリング
